### 색상
#### 파일 포멧
- 카테고리 속성 : d*   (모든 디렉토리)
  - dc 에 보면 속성탭이 있음. 거기에 표시 되는거 쓰면 됨.
  - 디렉토리는 d 로 시작.
  - 파일은 - 로 시작.

- 파일 색상 변경하기
  - 카테고리 속성은 비워 두면 됨. 속성에 따라 색깔을 다르게 할 경우 사용.  - 로 시작해야함.
```
- 카테고리 마스크
  - 구분자 ; 사용.
  
*.h;*.cpp
```
![디렉토리 색상 변경 바로 설정](./setting_color_file_filter.png)
